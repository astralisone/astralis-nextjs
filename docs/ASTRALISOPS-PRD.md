# AstralisOps v1 — Product Requirements Document (PRD)

## 1. Product Summary
AstralisOps is an AI-powered operations automation console for small businesses, service providers, and internal teams. It centralizes intake, scheduling, document processing, and workflow automation into a single dashboard and pipeline view.

## 2. Goals
- Automate 60–90% of operational busywork.
- Standardize intake → scheduling → document processing → fulfillment.
- Become the central nervous system for SMB operations and automation.
- Provide a foundation for marketplace-ready automation kits.

## 3. User Roles
- **Admin** – Owns configuration, pipelines, integrations, RBAC.
- **Operator** – Works queues/pipelines, processes leads, manages tasks.
- **Client (optional)** – External user submitting requests and documents.

## 4. Core Epics
- AI Intake & Lead Routing
- AI Scheduling
- AI Document Processing
- Dashboard
- Pipelines
- RBAC & Accounts
